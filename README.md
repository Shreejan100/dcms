# Dental Clinic Management System

A modern and responsive dental clinic management system built with PHP, MySQL, HTML, CSS, JavaScript, and Bootstrap.

## Features
- Role-based authentication (Admin, Dentist, Patient)
- Secure password hashing with bcrypt
- Real-time appointment scheduling
- Patient appointment management
- Dentist profile and schedule management
- Admin dashboard for overall system control
- Responsive design for all devices

## Requirements
- XAMPP Server
- PHP 7.4+
- MySQL 5.7+
- Modern web browser

## Installation
1. Clone this repository to your xampp/htdocs folder
2. Import the database schema from `database/dcms.sql`
3. Configure database connection in `config/database.php`
4. Access the application through your web browser at `http://localhost/dcms`
